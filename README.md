# Gene Expression Analysis Across Age and Condition

## Overview
This project analyzes gene expression patterns across different age groups and health conditions using Python.

## Features
- Condition-based comparison (Healthy vs Disease)
- Age group analysis (Young vs Old)
- Gene-wise expression trends
- Combined analysis of age and condition

## Tools Used
- Python (pandas, matplotlib)

## Motivation
To explore how biological data can be analyzed computationally to identify meaningful patterns related to disease and aging.

## Future Scope
- Use real gene expression datasets
- Add statistical testing
- Build interactive visualization dashboard
